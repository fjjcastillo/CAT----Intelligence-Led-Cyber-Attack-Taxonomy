 <h2 class="pt-3" id="detectable">
  Detection
 </h2>
 <b>
  Detectable by Common Defenses (Yes/No/Partial):
 </b>
 No
 <br/>
 <br/>
 <b>
  Explanation:
 </b>
 Defender will not know at first use what is valid or hostile traffic without more context.

 </b>
 Yes
 <br/>
 <br/>
