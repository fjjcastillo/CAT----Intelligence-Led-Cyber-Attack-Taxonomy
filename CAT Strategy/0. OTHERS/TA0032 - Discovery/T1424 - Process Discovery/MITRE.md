 <h1>
  Process Discovery
 </h1>
 <div class="row">
  <div class="col-md-8 description-body">
   <p>
    On Android versions prior to 5, applications can observe information about other processes that are running through methods in the ActivityManager class. On Android versions prior to 7, applications can obtain this information by executing the
    <code>
     ps
    </code>
    command, or by examining the
    <code>
     /proc
    </code>
    directory. Starting in Android version 7, use of the Linux kernel's
    <code>
     hidepid
    </code>
    feature prevents applications (without escalated privileges) from accessing this information
    <span class="scite-citeref-number" data-reference="Android-SELinuxChanges" id="scite-ref-1-a">
     <sup>
      <a aria-describedby="qtip-0" data-hasqtip="0" href="https://code.google.com/p/android/issues/detail?id=205565" target="_blank">
       [1]
      </a>
     </sup>
    </span>
    .
   </p>
  </div>
